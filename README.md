# Secret Santa for Kek

Secret Santa project for Kek, a beloved coder on ArtUnion.

## Overview
This project is a web page that has been customized as a Secret Santa gift for Kek.

## Requirements
- Docker
- Docker Compose (v1.27+ or Docker Compose v2)

## Quick start
1. Clone this repository
2. Start:
    - docker compose up
3. Access the app at http://localhost:34587
